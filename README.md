# [psr-7-framework]()
Demo framework

[![version][version-badge]][CHANGELOG]

## Run project

```shell script
composer serve
```
or
```shell script
php -S localhost:8000 -t public/
```

## Tests

```shell script
composer test
```
or
```shell script
./vendor/bin/phpunit
```
